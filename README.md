# Estudo de API

**Demo:** https://api-study-nine.vercel.app/

Exercícios práticos de JavaScript e Node.js para aprender backend do zero.

## Sobre

Este projeto contém uma série de exercícios progressivos focados em:

- Fundamentos de JavaScript (variáveis, funções, arrays, objetos)
- Manipulação de dados com map, filter, reduce
- Programação assíncrona (fetch, async/await, Promise.all)
- Módulos e organização de código

O objetivo é preparar desenvolvedores frontend para trabalhar com backend, com foco prático e direto.

## Estrutura

```
api-study/
├── index.html          # Menu de navegação
├── global.css          # Sistema de design
├── guia-exercicios.md  # Roteiro completo de estudos
├── utils/              # Funções utilitárias
│   └── converterMoeda.js
├── ex-1/               # Cartão de Visita Digital
├── ex-2/               # Calculadora de Orçamento
├── ex-3/               # Lista de Compras (map, filter, reduce)
├── ex-4/               # Catálogo de Filmes
├── ex-5/               # Validador de Email
├── ex-6/               # Gerador de Senhas
├── ex-7/               # Temporizador (setTimeout)
├── ex-8/               # Buscador de Piadas (fetch)
├── ex-9/               # Múltiplas Requisições (Promise.all)
└── ex-10/              # Biblioteca de Utilidades (módulos)
```

## Como usar

1. Clone o repositório

```bash
git clone https://github.com/seu-usuario/api-study.git
```

2. Abra o `index.html` no navegador ou use o Live Server do VS Code

3. Navegue pelos exercícios e acompanhe o progresso no `guia-exercicios.md`

## Tecnologias

- HTML5
- CSS3 (variáveis, grid, flexbox)
- JavaScript ES6+ (modules, arrow functions, destructuring)
- Fetch API
